## 对象的扩展
**1. 简洁表示：**
```js
let o = 1,k = 2;
let es5 = {
    o:o,
    k:k,
    methodes5:function(){ }
};
let es6 = {
    o,   //属性简写
    k,
    methodes6(){}   //方法简写
};
```
**2. 属性表达式：**
属性名可以使用表达式形式

**3. ES6中对象新增的api：**
- 可以通过`__proto__`读取或设置当前对象的prototype对象;
- 使用`Object.is({},{})`判断两个对象是否完全相等，类似===;
- `Object.assign(target, source1, source2)`合并对象；（1. 浅拷贝 2. 只包括对象自身可枚举的属性）
- `Object.entries(obj)` 遍历，返回键值对数组。（与for-in区别在于 for-in也枚举原型链中的属性）
